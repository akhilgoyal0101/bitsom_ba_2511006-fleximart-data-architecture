## Project Overview

This project implements an end-to-end data architecture solution for a fictional retail company, FlexiMart. It covers transactional data processing using ETL pipelines, analytical reporting using SQL, NoSQL database analysis with MongoDB, and dimensional modeling using a star schema for OLAP analytics.

The project demonstrates practical implementation of relational databases, NoSQL concepts, and data warehousing techniques aligned with real-world business scenarios.
